# 🌿 YungasVerdes

Sitio web de turismo en Jujuy desarrollado como práctica académica en la **Facultad de Ingeniería – Universidad Nacional de Jujuy**.  
Proyecto realizado en el marco de la materia **APU 2008 – Práctica: Desarrollo de un sitio web con Bootstrap, jQuery y Buenas Prácticas**.  
Fecha: 21-04-2026  

---

## 🎯 Objetivo
Desarrollar un sitio web moderno, interactivo y **100% responsive**, aplicando conceptos de **HTML5**, **Bootstrap** y **jQuery**, siguiendo buenas prácticas de diseño y optimización.  
El sitio se adapta correctamente a **Mobile, Tablet, Desktop y pantallas grandes**.

---

## 🗂️ Estructura del Sitio

### 1. Home
- Navbar responsive con dropdown/mega menú (Bootstrap)  
- Hero con imagen/video + texto animado (jQuery)  
- Cards de destinos con hover  
- Contador animado (jQuery)  
- Carousel de testimonios  
- Footer con formulario, redes sociales y mapa embebido  

### 2. Destinos
- Filtros dinámicos con jQuery (`.filter()`, `.hide()`, `.show()`)  
- Grid responsive (1, 2 o 3 columnas según pantalla)  
- Cards con efectos de zoom (CSS + jQuery)  
- Tabla de precios responsive (`table-responsive`)  

### 3. Agencias
- Cards con efecto flip (CSS + jQuery toggle)  
- Sistema de rating con estrellas (jQuery)  

### 4. Contacto
- Formulario con validación en tiempo real (`.on('input')`, `.val()`)  
- Spinner de carga (`spinner-border`)  
- Modal de confirmación  
- Diseño responsive con Flexbox + Grid de Bootstrap  

### 5. Precios
- Tabla comparativa  
- Hover dinámico  
- Tooltips (Bootstrap + jQuery)  

### 6. Blog
- Layout tipo revista  
- Filtro por categorías (jQuery)  
- Comentarios simulados  
- Animaciones al hacer scroll  

### 7. Módulo Educativo: Phishing
- Botón de **“Simulación de phishing”**  
- Modal Bootstrap con advertencia clara, ejemplos educativos y señales de fraude  
- Interacción: el usuario identifica errores y recibe feedback con jQuery  

---

## 🛠️ Tecnologías utilizadas
- **HTML5**  
- **CSS3** (Flexbox, Grid, efectos hover, sprite CSS)  
- **Bootstrap 5** (Navbar, botones, tooltips, modals, responsive grid)  
- **jQuery** (animaciones, filtros, validaciones, feedback dinámico)  
- **Font Awesome / Bootstrap Icons** (iconografía)  

---

## 🔒 Buenas Prácticas y Seguridad
- Validación de inputs con HTML5 (`required`, `pattern`) y jQuery  
- Sanitización básica de datos  
- Sprite CSS para optimizar carga de iconos (redes sociales, botones, iconos personalizados)  
- Módulo educativo de phishing (sin robo de datos reales)
